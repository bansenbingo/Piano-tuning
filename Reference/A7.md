## 1. 板卡概述

Nexys A7 是 Digilent 公司推出的基于 Xilinx Artix-7™ FPGA 的完整数字电路开发平台，是 **Nexys 4 DDR 的品牌重命名版本**（Nexys A7-100T 与 Nexys 4 DDR 功能完全相同）。

### 1.1 板卡演进历史

```
Nexys 4  →  Nexys 4 DDR  →  Nexys A7
（16MiB      （128MiB DDR2，   （新增 50T 变体，
 CellularRAM） 引脚重新布局）    100T 与 4DDR 相同）
```

**从 Nexys 4 迁移的关键差异**：
- 16 MiB Cellular RAM 替换为 128 MiB DDR2 SDRAM
- FPGA _bank_ 引脚布局改变
- **音频输出 (AUD_PWM) 需要开漏驱动**，而非 Nexys 4 的推挽驱动

### 1.2 版本区别

| 特性 | Nexys A7-100T | Nexys A7-50T |
|------|---------------|--------------|
| FPGA 型号 | XC7A100T-1CSG324C | XC7A50T-1CSG324**I** |
| 逻辑切片 (Slices) | 15,850 | 8,150 |
| LUTs | 63,400 | 32,600 |
| 触发器 | 126,800 | 65,200 |
| Block RAM | 1,188 Kb | 600 Kb |
| DSP 切片 | 240 | 120 |
| 时钟管理单元 | 6 | 5 |
| 工具链支持 | Vivado + ISE | **仅 Vivado** |
| 温度等级 | C（商业级） | **I（工业级）** |

> **注意**：50T 是工业级 (I)，100T 是商业级 (C)，这在选择器件时非常重要。

### 1.3 主要特性

- **FPGA 内部特性**：每切片 4 个 6 输入 LUT + 8 个触发器；内部时钟 >450 MHz；**双通道 1 MSPS XADC (12 位 ADC)**
- **内存**：128MiB DDR2 + 128Mib 串行 Flash + microSD 卡槽
- **通信**：10/100 以太网 PHY、USB-JTAG、USB-UART、USB HID 主机
- **用户 I/O**：16 拨码开关、16 LED、2 RGB LED、8 位 7 段数码管、6 按钮
- **音视频**：12 位 VGA（4096 色）、PWM 音频、PDM 麦克风
- **传感器**：3 轴加速度计、温度传感器
- **扩展**：4 个 Pmod + 1 个 XADC Pmod（共 32 个 FPGA I/O）

## 4. 功能模块详解

### 4.1 XADC（内部模数转换器）

- **双通道 12 位 ADC，1 MSPS**
- 通过 JXADC 接口接收差分模拟输入
- 通过**动态重配置端口 (DRP)** 控制访问
- 可监测各电源轨电压和 FPGA 内部温度
- 参考文档：*7 Series FPGAs and Zynq-7000 XADC Dual 12-Bit 1 MSPS ADC*

### 4.2 温度传感器 ADT7420 详解

| 参数 | 值 |
|------|-----|
| 分辨率 | 16-bit |
| 精度 | ±0.25°C |
| I²C 地址 | 0x4B（1001011） |

**寄存器**：
- TLOW: 0x06:0x07
- THIGH: 0x04:0x05
- TCRIT: 0x08:0x09

**开漏输出**：INT 引脚（超出 TLOW/THIGH 范围）、CT 引脚（超过 TCRIT 临界值），使用时**需 FPGA 内部上拉**。

**快速温度读取**：
```
1. 上电默认指向温度 MSB 寄存器
2. 读 2 字节（MSB + LSB）→ 补码 16 位整数
3. 右移 3 位，乘以 0.0625 = 摄氏温度
```

**I²C 读寄存器流程**：写目标寄存器地址 → 发 I²C 重启 → 发读请求。**若无重启条件，地址寄存器会被复位为 0x00**。

### 4.3 加速度计 ADXL362 详解

| 参数 | 值 |
|------|-----|
| 分辨率 | 12-bit（另提供 8-bit 格式） |
| 量程 | ±2g / ±4g / ±8g |
| 灵敏度 | 1 mg/LSB（±2g 量程） |
| 功耗 | <2μA @100Hz；运动唤醒 270nA |
| SPI 模式 | Mode 0（CPOL=0, CPHA=0） |
| SPI 时钟 | 1-5 MHz |

**特点**：不使用欠采样，全带宽采样。测量模式下持续测量 X/Y/Z 数据。中断可映射到 INT1/INT2，**需 FPGA 内部上拉**。

### 4.4 PDM 麦克风详解

| 参数 | 值 |
|------|-----|
| SNR | 61 dBA |
| 灵敏度 | -26 dBFS |
| 频率响应 | 100Hz - 15kHz |
| 时钟频率 | 1-3.3 MHz（典型 2.4MHz） |

**PDM 原理**：通过 Delta-Sigma 调制，1=正脉冲，0=负脉冲。全 1=最大正幅度，全 0=最小幅度。**flip-flop 输出的平均值等于输入模拟信号值**，用计数器统计一定时间内的 1 的个数即可得到模拟值。

**L/R 选择**：
- M_LRSEL 低 → 时钟**上升沿**数据有效
- M_LRSEL 高 → 时钟**下降沿**数据有效

**采样示例**：2.4MHz 时钟，7 位精度 24KHz，可用两个计数器在 12KHz 下计数 128 个样本。

### 4.5 单声道音频输出详解

- **Sallen-Key Butterworth 4 阶低通滤波器**
- 输入 AUD_PWM 连接 FPGA **A11** 引脚
- 滤波器截止频率：fc1=12.5KHz（G=1），fc2=16.3KHz（G=1）
- **必须开漏驱动**：逻辑 0 驱动低，逻辑 1 高阻（由板载上拉到干净的模拟 3.3V 建立）
- 低通滤波器作为重构滤波器，将 PWM 数字信号转为模拟电压

**AUD_PWM 引脚**：A11（_bank_ 35）
**AUD_SD（关断）引脚**：D12（_bank_ 16）

**PWM 原理**：低通滤波器 3dB 频率应比 PWM 频率低一个数量级。若音频含 5KHz 信息，PWM 频率应至少 50KHz。输出电压幅度/Vdd = 占空比。

## 6. FPGA 配置方法

### 6.1 四种配置方式

| 方式 | 端口 | 特性 |
|------|------|------|
| USB-JTAG | J6 (PROG) | 易失，随时可编程 |
| Quad-SPI Flash | 内部 | 非易失，上电自动加载（Master SPI） |
| MicroSD 卡 | J1 | FAT32，根目录.bit 文件 |
| USB 闪存盘 | J5 | FAT32，根目录.bit 文件 |

### 6.2 跳线设置

#### JP1 编程模式

| 位置 | 模式 |
|------|------|
| JTAG | 仅 JTAG 编程 |
| QSPI | 从 SPI Flash 加载（Master SPI） |
| USB/SD | 从 USB 或 SD 加载 |

#### JP2 介质选择

| 位置 | 介质 |
|------|------|
| SD | MicroSD 卡 |
| USB | USB 闪存盘 |

### 6.3 位流信息

- **Artix-7 100T 位流典型大小：30,606,304 位**
- 未压缩 JTAG 编程约 5 秒
- **启用压缩后压缩比可达 10:1**
- Flash 编程约 4-5 分钟（主要是擦除耗时），但配置后加载 <1 秒
- Nexys A7 支持 x1/x2/x4 总线宽度，QSPI 数据率最高 50 MHz

### 6.4 BUSY LED (LD21) 状态含义（FPGA 未编程时）

| LED 状态 | 含义 |
|----------|------|
| 常亮 | 辅助 MCU 启动中或正在读取配置介质并下载 |
| 慢速脉冲 | 等待插入配置介质 |
| 快速闪烁 | 配置出错 |

> FPGA 成功配置后，LD21 行为由应用决定（如接 USB 键盘时收到 HID 报告会快闪）。

### 6.5 USB/SD 配置步骤

```
1. 存储设备格式化为 FAT32
2. 根目录放单个.bit 文件
3. 插入存储设备
4. JP1 设为 USB/SD
5. JP2 选择介质
6. 按 PROG 或重新上电
```

> 非目标 Artix-7 器件的.bit 文件会被 FPGA 拒绝。

## 7. 时序与协议详解

### 7.1 VGA 640×480@60Hz 时序（25MHz 像素时钟）

| 参数 | 水平（像素） | 垂直（行） |
|------|------------|-----------|
| 显示区 | 640 | 480 |
| 前肩 (Front Porch) | 16 | 10 |
| 同步脉冲 (Sync) | 96 | 2 |
| 后肩 (Back Porch) | 48 | 33 |
| **总计** | **800** | **525** |

- 刷新频率：50Hz-120Hz（典型 60Hz）
- HS/VS 同步脉冲电平：3.3V（或 5V）
- 信息仅在正向扫描时显示

### 7.2 PS/2 键盘/鼠标协议

**HID 主机**：PIC24 隐藏 USB HID 协议，仿真 PS/2 总线。FPGA 作为主机，MCU 作为 PS/2 设备。可复用现有 PS/2 IP 核。

**数据格式**：11 位字 = 1 起始位(0) + 8 数据位(LSB first) + 1 奇校验位 + 1 停止位(1)。时钟 20-30KHz，数据在时钟**下降沿**有效。

**设备识别**：
- 连接时发送 "self-test passed" (0xAA)
- Read ID 命令 (0xF2) 读设备 ID
- 鼠标在 0xAA 后发送 ID (0x00) 以区分键盘

**键盘扫描码**：
- 按下发送扫描码，按住每约 100ms 重复
- 释放发送 F0 + 扫描码
- 扩展键前缀 E0；释放发 E0 F0 + 扫描码

**键盘命令**：

| 命令 | 动作 |
|------|------|
| ED | 设置 Num/Caps/Scroll Lock LED |
| EE | Echo 测试 |
| F3 | 设置扫描码重复率 |
| FE | 重发最近扫描码 |

**鼠标数据**：每次移动发 3 个 11 位字（共 33 位）。相对坐标系：右=X 正，左=X 负，上=Y 正，下=Y 负。XS/YS=符号位，XV/YV=溢出位，L/R=左右键。连续移动约每 50ms 重复。

**鼠标命令**（IntelliMouse 扩展）：

| 命令 | 动作 |
|------|------|
| EA | 设置流模式 |
| F4 | 使能数据报告 |
| F5 | 禁止数据报告 |
| F3 | 设置采样率 |
| FE | 重发上个包 |

> **注意**：PS/2 信号为开漏，使用时 FPGA 内需启用数据/时钟引脚上拉。

### 7.3 DDR2 MIG 配置参数

| 设置 | 值 |
|------|-----|
| 内存类型 | DDR2 SDRAM |
| 最大时钟周期 | 3000ps (667Mbps) |
| 推荐时钟周期 | 3077ps (650Mbps) |
| 内存型号 | MT47H64M16HR-25E |
| 数据位宽 | 16 |
| 数据掩码 | 启用 |
| 片选 | 启用 |
| Rtt(ODT) | 50Ω |
| 内部 Vref | 启用 |

> 因时钟生成原语限制，推荐用 650Mbps 而非理论最大 667Mbps。可用 Digilent 提供的 DDR-to-SRAM 适配模块简化开发（牺牲带宽换简单），或用 MIG 生成 Native/AXI4 接口。
